## 安装：
Steam版本在以后开了创意工坊后直接订阅即可，没有创意工坊时，在游戏根目录下创建一个`mods`文件夹，将下载的安装包解压到里面即可

安装完成后的目录结构如下：

```sh
-- Slay the Spire 2
    |-- SlayTheSpire2.exe
    |-- mods
         |-- JmcModLib
         |-- QuickSL
              |-- QuickSL.dll
              |-- QuickSL.pck
              |-- QuickSL.json
```

第一次使用Mod的用户，需要将旧存档迁移到新的存档中去，**否则以前的记录会消失**，建议使用我的[存档槽Mod](https://github.com/JMC2002/SlayTheSpire2_BetterSaveSlots)做这个操作。
