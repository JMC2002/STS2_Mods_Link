I have made quite a few small mods myself. Since they share a prerequisite mod, many users have said that the prerequisite is hard to find, and many people are not familiar with GitHub either. ## Installation

For the Steam version, you will be able to subscribe directly through the Steam Workshop once it is available. Before Workshop support is available, create a `mods` folder in the game root directory, then extract the downloaded package into it.

After installation, the directory should look like this:

```sh
-- Slay the Spire 2
    |-- SlayTheSpire2.exe
    |-- mods
         |-- JmcModLib
         |-- QuickSL
              |-- QuickSL.dll
              |-- QuickSL.pck
              |-- QuickSL.json
```

If this is your first time using mods, you need to migrate your old saves into the new save location. **Otherwise, your previous records will disappear.** I recommend using my [Better Save Slots mod](https://github.com/JMC2002/SlayTheSpire2_BetterSaveSlots) for this operation.

## Mod Links

This section lists the Release links for the mods I have published so far. Open the link and download the compressed package in `zip` format. Each subsection includes a short introduction. For detailed features, check the README on that mod's homepage.

### Prerequisite Mod

> All of my mods require this mod. Other modders are also welcome to use it. See its README for detailed features.

[JmcModLib](https://github.com/JMC2002/SlayTheSpire2_JmcModLib/releases)

### Quick Save/Load

> A quick save/load mod with hotkey binding support, including key combinations, controller binding support, and multiplayer support where both host and clients can trigger quick save/load.

[Quick Save/Load](https://github.com/JMC2002/SlayTheSpire2_QuickSL/releases)

### Better Save Slots

> Customize the number of save slots, with Steam Cloud sync support. Also supports copying and pasting saves between different slots, plus one-click import for normal unmodded saves.

[Better Save Slots](https://github.com/JMC2002/SlayTheSpire2_BetterSaveSlots/releases)

### All Cards Become One Card

> Turns every card into a specific card.

[All Cards Become One Card](https://github.com/JMC2002/SlayTheSpire2_AllCardIs/releases)

### Multiplayer Anti-console Cheating

> As the host, you can prevent clients from using the console. As a client, you can detect who used the console.
